Ayuda de Reservas

Bienvenido a la sección de ayuda de reservas del club social. Aquí encontrarás información sobre cómo utilizar las diferentes funcionalidades relacionadas con las reservas.

Seleccionar Instalación

Para seleccionar una instalación, utiliza el combo box "Selecciona una Instalación". Esto actualizará las horas disponibles para la instalación seleccionada.

Seleccionar Fecha

Para seleccionar una fecha, utiliza el selector de fecha. Esto actualizará las horas disponibles para la fecha seleccionada.

Seleccionar Socio

Para seleccionar un socio, utiliza el combo box "Selecciona un Socio". Puedes buscar por nombre o número de socio.

Horas Disponibles

Las horas disponibles para la instalación y fecha seleccionadas se mostrarán en la cuadrícula. Las horas reservadas aparecerán deshabilitadas.

Realizar Reserva

Para realizar una reserva, selecciona una hora disponible y confirma la reserva en el cuadro de diálogo que aparece.

Imprimir Reporte de Reservas

Para imprimir un reporte de las reservas, haz clic en el botón "REPORTE RESERVAS".

Icono de Ayuda

Haz clic en el icono de ayuda en la esquina inferior derecha para abrir esta ventana de ayuda en cualquier momento.
